# v0 PMS Dashboard - Branch Strategy

## v0 Configuration: Always Push to Main

**IMPORTANT**: v0 (and all developers) MUST work exclusively on `main` branch.

- v0 creates all changes directly on `main`
- v0 commits all changes to `main`
- v0 pushes all changes to `origin/main`
- **NO feature branches**
- **NO pull requests**
- Direct push to `main` = automatic Vercel deployment

### Workflow

```bash
# 1. Make changes locally
git checkout main
# ... edit files ...

# 2. Commit changes
git commit -m "type: descriptive message"

# 3. Push to GitHub
git push origin main

# 4. Vercel automatically deploys when main is updated
```
